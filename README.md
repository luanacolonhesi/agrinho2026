# agrinho2026
📌 Sobre o projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, com o tema:

“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”

A aplicação tem como objetivo apresentar uma página web informativa e interativa sobre um tema específico dentro da sustentabilidade no meio rural, mostrando como é possível conciliar produção agrícola e preservação ambiental.

🎯 Tema escolhido

"Construa um agro forte e sustentável"

💡 Objetivo
Informar sobre práticas sustentáveis na agricultura
Demonstrar a importância do equilíbrio entre produção e meio ambiente
Utilizar tecnologias web para criar uma experiência interativa
Incentivar a conscientização ambiental no meio rural e urbano
🛠️ Tecnologias utilizadas
HTML5 → estrutura da página
CSS3 → estilização e layout
JavaScript → interatividade
Inteligência Artificial → apoio na criação e organização do projeto
